# Room Build Action Report — Execution Guide

## Overview

The **Room Build Action Report** turns intelligence into action. It's a complete execution document that tells you:

1. **Who to invite first** (Top 30 with priority scores)
2. **Who to source** (Gap lists for investors, CEOs, regions, sectors)
3. **Where to search** (Apollo search briefs with exact criteria)
4. **What the ideal room looks like** (Target balance across all dimensions)

**No more guessing. This is your sourcing playbook.**

---

## What Gets Generated

### 1. Top 30 Women to Invite First

**Ranked by priority score** combining:
- Influence score (heavily weighted)
- Speaker potential (+3 points)
- Sponsor intro potential (+2 points)
- Type (Founder/CEO +3, Investor +2)
- Regional bonus (+1 if not London)

**For each woman:**
| Column | What It Means |
|--------|---------------|
| **Name** | Full name |
| **Company** | Where she works |
| **Sector** | Automatically classified |
| **Region** | Geographic location |
| **Influence Score** | 0-10 calculated |
| **Why She Strengthens** | Specific reasons (e.g., "High influence, CEO credibility, Speaker potential") |
| **Suggested Invite Angle** | Exact invitation pitch |

**Example:**
```
Anne Boden | Starling | Finance | London | 9/10
Why: High influence, CEO credibility, Speaker potential
Invite Angle: "Feature speaker role + host table"
```

**Action:** Start here. These are your confirms.

### 3. Apollo Search Briefs

**Ready-to-paste search criteria** for each gap category:

Each brief includes:
- **Job Titles** (exact titles to search)
- **Sectors** (industry focus)
- **Locations** (geography)
- **Keywords** (search terms)
- **Company Types** (size/stage)
- **Target Count** (how many to find)
- **Notes** (where to source)

```

Sectors: Venture Capital, Private Equity, Finance, Fintech
Locations: London, UK
Keywords: investor, venture, capital, private equity, angel, VC
Company Types: Venture Capital, Private Equity, Investment Firm
```

## How to Use the Report

### Step 1: Generate the Report

**On the Intelligence Dashboard:**

Click "Room Build Action Report" section → Choose format:
- **View Report** — See data in browser
- **Download Markdown** — Pretty formatted document (for sharing)
- **Download CSV** — Spreadsheet for tracking (for Sheets/Excel)

### Step 2: Export & Share

**Markdown (.md)**
- Best for: Sharing with team, reading on GitHub, pasting into docs
- Use: Documentation, planning meetings, shared workspace
- Open in: Any text editor, markdown viewer, GitHub, Notion

**CSV (.csv)**
- Best for: Tracking, building sourcing lists, updating in spreadsheets
- Use: Google Sheets, Excel, CRM uploads
- Columns: Name, Company, Sector, Region, Score, Why, Angle

### Step 3: Execute

**Top 30 Women:**
1. Copy list to CRM or email
2. Personalize invite angles
3. Send warm introductions (via sponsor/ally)
4. Track responses in Guest database → Update Status

**Investor Gap:**
1. Open Apollo
2. Paste search criteria from brief
3. Add filters: Female, UK-based
4. Save 8-10 qualified leads
5. Build intro strategy (LinkedIn, board members, angels)

**CEO Gap:**
1. Search Apollo with CEO brief
2. Prioritize FTSE, fast-growth, founded-by-women companies
3. Aim for 4-6 qualified candidates
4. Track in Organisation database

**Regional Gap:**
1. Search Scottish / regional business awards
2. Search LECs, female founder networks
3. Aim for 5 strong regional leaders
4. Verify they're not London-based

**Healthcare Gap:**
1. Search NHS England networks
2. Search healthcare awards (HSJ, etc.)
3. Target 5 NHS Trust CEOs or healthcare leaders
4. Aim for mix of public/private sector

**Education Gap:**
1. Search Russell Group universities (Vice Chancellors)
2. Search independent school heads
3. Target 5 education leaders
4. Reach out via university networks

**Corporate Gap:**
1. Search large employers (FTSE, major retailers, banks)
2. Focus on Chief People Officers, HR leaders
3. Target 5 companies with 5000+ employees
4. Use Employer of Choice awards as reference

### Step 4: Track Progress

**Update Guest Database as you:**
1. Identify new candidates → Add to Guests, set Status = "Identified"
2. Invite candidates → Update Status = "Invited"
3. Get confirmations → Status = "Confirmed"
4. Collect payments → Status = "Paid"

**Dashboard updates in real-time:**
- Progress towards 100 guest target
- Sector distribution changes
- Region breakdown updates
- Role type distribution

**Compare to Room Balance Target:**
- Are you hitting sector targets?
- Are you hitting region targets?
- Are you hitting type targets?
- Adjust sourcing if needed

### Step 5: Recommendations Panel

Dashboard also shows:
- **Gaps still to fill** (e.g., "Only 2 investors, need 8")
- **Imbalances detected** (e.g., "Finance is 18% of room")
- **Opportunities** (e.g., "22 women can introduce sponsors")

**Use this to adjust sourcing in real-time.**

## Troubleshooting

### Q: Report says "8 investors needed" but I have 5?
**A:** Report always regenerates from current database. As you add guests, gap numbers change automatically.

### Q: Can I edit the search criteria?
**A:** Yes! The Apollo briefs are templates. Modify:
- Job titles (add/remove)
- Sectors (add/remove)
- Locations (adjust by region)
- Keywords (customize)

### Q: Why isn't my new guest showing up?
**A:** Report regenerates on each call. Make sure guest is saved in database with:
- Name ✓
- Company ✓
- Role (auto-classified) ✓
- Status = "Identified" or higher ✓

### Q: Can I download just the top 30?
**A:** CSV format includes all sections. Filter in Excel/Sheets if needed.

### Q: How often should I regenerate?
**A:** Weekly is good. More often if sourcing actively.
