On sign-off: lock items read-only; store signedOffBy, signedOffAt.

8) Acceptance criteria

Creating a new event auto-populates phases with the seeded activities.

Planner can add/edit/delete tasks; assign owners; set dates; upload files; change status.

Phase progress % updates live; Mark Phase Complete locks items and records sign-off.

Filters/search/saved views work across the board, Kanban, and Timeline.

All data persists to the existing backend tied to eventId.

No color or theme changes.
