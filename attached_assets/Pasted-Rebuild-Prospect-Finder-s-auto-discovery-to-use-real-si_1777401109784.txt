============================================================

Create scripts/discovery/anniversary-scanner.ts.

Logic:
- Use COMPANIES_HOUSE_API_KEY (UK) and a Nigerian equivalent if one exists (CAC API — check if available, otherwise leave a TODO)
- Daily scan: identify UK companies whose incorporation date matches a milestone anniversary in the next 90 days. Milestones: 5, 10, 15, 20, 25, 30, 40, 50 years.

- Filter: only company types worth pursuing (LTD, PLC, LLP — not charity-only or sole trader by default; flag charities separately for a charity-events angle later)

  - country='UK'

- Schedule: daily at 06:00 UK time (so the 7:30am morning job picks them up the same day)
- Idempotency: don't insert duplicates — check on (company_number, milestone_year) unique pair

Quotas (Companies House free tier is 600 calls per 5 minutes):
- Throttle to max 500 calls per scan
- Persist scan progress so it can resume mid-scan if it hits quota
- Log how many candidates found per day in a discovery_runs table
