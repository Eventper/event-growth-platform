FIX 7 — CONNECT CONTRACTS FINDER API:

In server/saas-tender-routes.ts update the 
search function:

Contracts Finder API:
GET https://www.contractsfinder.service.gov.uk/Published/Notices/PublicSearch/Search
Parameters:
  Keyword: from user search
  NoticeType: 1 for contract notices, 2 for awards
  PostedFrom: publication date from
  PostedTo: publication date to
  RegionCodes: array of UK region codes
  CPVCodes: sector codes
  Page: pagination
  PageSize: 20

Parse response and normalise to standard format:
{
  id, title, description, buyer, value, 
  value_estimated, published_date, deadline,
  status, source, region, category, url,
  cpv_codes, procedure_type, sme_suitable,
  winner, award_date, award_value
}

Find a Tender OCDS API:

Headers: CDP-Api-Key: [key]
Parameters:
  q: keywords
  published_from: date
  published_to: date
  status: active/complete/cancelled
  region: ITL code

Parse OCDS format and normalise to same
