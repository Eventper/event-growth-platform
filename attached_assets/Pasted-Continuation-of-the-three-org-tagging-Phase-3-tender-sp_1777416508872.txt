E2. CONFIDENCE SCORING

After section generation, run a SECOND AI pass:

  "Score this bid section 0-100 on three dimensions:
  (a) Coverage of mandatory requirements from the tender — score how many extracted requirements are addressed in this section, weighted by importance (mandatory pass/fail counts more than nice-to-have)
  (b) Evidence strength — count direct citations to Bid Vault evidence and learning vault past wins. More cited evidence = higher score
  (c) Voice alignment — match against EP's known voice from past wins (CEFAS, Lloyds banking, DEFRA portfolio governance). Generic AI-sounding language = lower score
  
  Return JSON: { coverage_score, evidence_score, voice_score, overall_confidence, weak_points: [...] }"

Add columns to bid section table:
- coverage_score INT
- evidence_score INT
- voice_score INT
- overall_confidence INT
- weak_points TEXT[]
- last_scored_at TIMESTAMP

Surface in the section UI:
- Traffic light next to section header: green 80+, amber 50-79, red <50
- Tooltip on hover shows the three sub-scores
- "What would improve this section?" button below each section reveals weak_points
- Clicking a weak point links to suggested action

E3. GAP ANALYSIS

After Generate-All-16 (or manually triggered), run a final AI pass:

  "Compare the generated bid against extracted tender requirements.
  
  For each requirement marked mandatory_pass_fail in extracted_facts: is it explicitly addressed in any of the 16 sections? Quote where.
  
  For each evaluation_criterion: is it strongly addressed (multiple citations, specific evidence) or weakly (generic claims, no evidence)?
  
  Return JSON: { gaps: [{requirement, severity: 'critical'|'high'|'medium', suggested_section, suggestion}], coverage_summary }"

Store as bid_gaps table:
- id, tender_id, requirement_id, severity, suggested_section_id, suggestion
- status ('open'|'resolved'|'ignored'), created_at

- Expandable list shows each gap with: requirement, severity, which section should address it, click jumps to that section
- Per gap: [Mark resolved] [Ignore] [Add evidence]
