STEP 1 — EP TEAM PROPOSES SLOTS
In the staff portal project page for ALLI Foundation add a Schedule Gateway Meeting button on each phase milestone.

- Select meeting: Gateway 1, 2, or 3
- Select 6 to 8 available time slots across the relevant week
- Each slot: date, start time, duration
- Add agenda for the meeting
- Click Send to All Attendees

STEP 2 — ATTENDEES SELECT AVAILABILITY
Public scheduling page at /meeting/alli-gateway-[1/2/3] — no login required

Page shows:
- Meeting title and agenda
- List of all proposed time slots as selectable cards
- Each slot shows: date, day of week, time
- Attendee enters their name
- Clicks green tick on slots that work — can select multiple
- Clicks amber question mark on slots that might work
- Clicks red X on slots that do not work
- Submits availability

Page updates in real time showing:
- How many people have responded
- Which slots have most green ticks
- Colour coding: dark green — everyone available, light green — most available, amber — some available, red — conflicts

STEP 3 — AUTO SELECTION
When all attendees have responded OR deadline passes:
- System automatically selects the slot with most green ticks
- If tie — select earlier slot

Send confirmation email to all attendees.
